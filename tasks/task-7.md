# Task 7 ▸ React 前端
# 预估：4-5 小时
# ─────────────────────────────────────

## 目标
最小可用的 Web 界面：应用桌面 + AI 对话生成 + 生成进度 + 应用卡片列表。

## 参考文件
@CLAUDE.md

## 要做的事

用 Vite + React 19 + TypeScript + Tailwind CSS 搭建前端。

1. `web/` — Vite 项目初始化
2. 页面：
   - `Setup.tsx` — 首次设置向导（创建管理员）
   - `Desktop.tsx` — 应用桌面（主页面）
3. 组件：
   - `AIPanel.tsx` — AI 对话面板（底部输入框，支持 SSE 流式进度展示）
   - `AppCard.tsx` — 应用卡片（名称、状态指示灯、启停按钮、打开链接）
   - `Progress.tsx` — 生成进度条（6 个步骤：意图→生成→审查→构建→部署→完成）
4. 状态管理：
   - `stores/authStore.ts` — token 管理
   - `stores/appStore.ts` — 应用列表
5. API：
   - `api/client.ts` — fetch 封装 + JWT 注入 + SSE 处理

## UI 要求

- 深色主题，简洁
- 桌面页面：顶部系统状态栏（CPU/内存），中间应用网格，底部 AI 输入框
- AI 面板：输入框 + 生成进度（步骤高亮 + 耗时）+ 完成后自动刷新应用列表
- 应用卡片：🟢运行中/🔴已停止 状态灯，点击打开应用 URL

## 验收

```
浏览器打开 http://localhost → 设置向导 → 创建用户 → 登录
→ 看到应用桌面（空）
→ 底部输入"做一个待办事项应用"
→ 看到 6 步进度条逐步完成
→ 完成后桌面出现应用卡片
→ 点击卡片打开应用
```
